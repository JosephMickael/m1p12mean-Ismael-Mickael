## Configuration

1. Copier le fichier `.env.example` en `.env` :

```bash
cp backend/.env.example backend/.env
```

2. Go to the frontend & backend directories

```bash
  cd frontend
  cd backend
```

3. Install dependencies

```bash
  npm install
```

4. Start the server (each)

```bash
  npm start
```

**Client:** Angular, Bootstrap

**Server:** Node, Express

**Database:** MongoDB
